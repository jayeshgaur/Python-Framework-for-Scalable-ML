API Reference
=============

The API Reference provides documentation for the Simple Classifier library's classes and modules.

- Splitters: Documentation for the classes and modules related to dataset splitting strategies.
- Splitter Datasets: Documentation for the classes and modules providing predefined datasets for training and testing classifiers.
- Classifiers: Documentation for the classes and modules related to classification algorithms.
- Hyperparameter Tuner: Documentation for the classes and modules for tuning hyperparameters of classifiers.
- Classifier Profilers: Documentation for the classes and modules for profiling and evaluating classifiers.

Please refer to the respective sections in the API Reference for detailed information on each topic.
